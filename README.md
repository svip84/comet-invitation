## 🚀 2. 注册流程

### 步骤 1：获取邀请码
请联系已有用户获取邀请码或注册链接。

### 步骤 2：访问注册页面
1. 打开 Comet 官方网站
2. 点击「注册」按钮
3. 输入您的邮箱地址

### 步骤 3：填写邀请码
在注册页面输入您获得的邀请码。

### 步骤 4：完成注册
1. 设置您的账户密码
2. 填写必要的个人信息
3. 同意服务条款
4. 点击「完成注册」

### 步骤 5：验证邮箱
检查您的邮箱，点击验证链接激活账户。

## ❓ 4. 常见问题与联系方式

### 常见问题

#### Q1: 邀请码有使用期限吗？
A: 是的，每个邀请码都有特定的有效期，请在有效期内完成注册。

#### Q2: 一个邀请码可以使用多次吗？
A: 通常情况下，每个邀请码只能使用一次。部分特殊邀请码可能支持多次使用。

#### Q3: 如果邀请码无效怎么办？
A: 请检查邀请码是否输入正确，或联系邀请人获取新的邀请码。

#### Q4: 注册需要付费吗？
A: 请参考 Comet 官方网站的最新定价信息。部分功能可能提供免费试用。

#### Q5: 支持哪些支付方式？
A: Comet 通常支持信用卡、借记卡等主流支付方式，具体以官网为准。
