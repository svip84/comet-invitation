## 🚀 2. 注册流程
### 步骤 1：获取邀请码
请联系已有用户获取邀请码或注册链接。
### 步骤 2：访问注册页面
1. 打开 Comet 官方网站
2. 点击「注册」按钮
3. 输入您的邮箱地址
### 步骤 3：填写邀请码
在注册页面输入您获得的邀请码。
### 步骤 4：完成注册
1. 设置您的账户密码
2. 填写必要的个人信息
3. 同意服务条款
4. 点击「完成注册」
### 步骤 5：验证邮箱
检查您的邮箱，点击验证链接激活账户。
